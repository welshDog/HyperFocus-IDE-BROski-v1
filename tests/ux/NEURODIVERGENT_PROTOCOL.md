# Neurodivergent Developer Testing Protocol

## 🎯 Objective
Assess the cognitive load, clarity, and usability of the HyperSwarm Control Center for developers with ADHD, Autism, and Dyslexia.

## 📋 Participant Profile
- **Total**: 3-5 Developers
- **Background**: Mid-to-Senior level
- **Traits**: Self-identified as neurodivergent (ADHD, Autism, Dyslexia)

---

## 🧪 Test Scenarios

### Scenario 1: Intent Execution (Focus & Clarity)
**Task:** "You found a bug in the payment API. Use the Intent Box to fix it."
- **Expected Flow**: Locate box -> Type intent -> Click Execute.
- **Success Criteria**: < 30 seconds.
- **Metrics**:
    - Time to locate input.
    - Distraction points (eye tracking/observation).

### Scenario 2: Status Identification (Visual Processing)
**Task:** "Which agent is currently working on a task?"
- **Expected Flow**: Scan graph/list -> Identify 'active' status or color.
- **Success Criteria**: 100% Accuracy.
- **Metrics**:
    - Errors in identification.
    - Reliance on color vs. text (colorblind check).

### Scenario 3: Safety Configuration (Cognitive Load)
**Task:** "Set the Safety level to 20%. What does the system tell you?"
- **Expected Flow**: Locate slider -> Drag -> Read feedback.
- **Success Criteria**: 100% Comprehension of "⚠️ DANGER".
- **Metrics**:
    - Reaction time to feedback.
    - Frustration level (slider precision).

### Scenario 4: Budget Check (Information Retrieval)
**Task:** "How much money have we spent today?"
- **Expected Flow**: Locate 'Cost Tracker' card -> Read 'Today' value.
- **Success Criteria**: < 5 seconds.
- **Metrics**:
    - Search pattern (random vs. structured).

## 🗣️ Post-Test Interview Guide

**1. Clarity (1-5):**
"How easy was it to understand what to do?"
- *Why did you give that score?*

**2. Calmness (1-5):**
"Did the interface feel overwhelming or calm?"
- *Did any specific element distract you?*

**3. Control (1-5):**
"Did you feel in charge of the agents?"

**4. Daily Use:**
"Would you use this tool every day? Why/Why not?"

**5. Magic Wand:**
"If you could change one thing to make this better for your brain, what would it be?"
